PALFAM









ABOUT

PALFAM uses cloud-based directory to compare the faces captured during account creating for each login or sign up. This achieves the important security key triad confidentiality. As a Multi-Factor Authentication(MFA) along with face recognition, we ask the security questions and in additional we send a Timed OTP(TOTP) to users mobile. The combination of face recognition, security questions and OTP makes a secured application with MFA.
